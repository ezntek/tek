ChromiumOS Daily Builds

Hello and Welcome! Here I will provide ChromiumOS Daily Builds/IMGs for you to use and download. These are updated DAILY and there are NO autoupdates so pick a build and use it. If you wanna upgrade then reflash a new version.

DO NOT use this as a daily-driver unless you are very adventurous. These are rolled right off the trunk so they are F--- Unstable.

You can use BalenaEtcher (DOWNLOAD HERE) or DD from a linux terminal to write it to a USB Stick or use a live linux distro with the image on it and flash it that way. Downloads are in .tar.lzma so you need something like 7-zip on Windows or TheUnarchiver on macOS
